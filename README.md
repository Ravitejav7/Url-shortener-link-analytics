# URL Shortener

A Spring Boot Java project scaffolded with:

- Java 21
- Spring Boot 4.1.0
- Maven 3.9.16

## Run tests

```powershell
.\mvnw.cmd test
```

This project includes a local Maven command, so you do not need global `mvn` to be working.

## Start the app

```powershell
.\mvnw.cmd spring-boot:run
```

Then open:

```text
http://localhost:8080/
```

You should see a JSON response saying Spring Boot is running.
